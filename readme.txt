Steps to install this program.

1) Run the following command on command prompt.
    'npm install'
2) Start Express server by executing following command.
    'node server.js'
3) Open any browser and go to 'localhost:3000'.


Procedure of App.

1) You have to register a train from 'ADMIN PANEL'.
2) After registering a train, You will see a train will be listed in 'Check Availability' Page from where we can book ticket.
